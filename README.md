## How to run local
```bash
npm install
```
```bash
npm run start:dev
```
API will live on
```bash
http://localhost:3001/
```
url : http://localhost:3001/

## How to run with Docker
```bash
docker-compose build
```
```bash
docker-compose up -d
```
API will live on
```bash
http://localhost:3000/
```
url : http://localhost:3000/
